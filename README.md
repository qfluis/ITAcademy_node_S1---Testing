# ITAcademy_node_S1---Testing
Para instalar dependencias(Jest):
```
npm install
```
Para ejecutar todos los tests:
```
npm run test
```
Para ejecutar las suites de test separadas por nivel:
```
npm run test_n1
npm run test_n2
npm run test_n3
```
